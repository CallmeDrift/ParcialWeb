# ParcialWeb

Parcial de web, noticiero sobre la jornada de proyectos integradores 2025-2. Este README explica cómo instalar dependencias, ejecutar la aplicación y cómo ejecutar los tests (unitarios y e2e).

# Requisitos para ejecutarlo

- Node.js (v18+ )


# Instalación

Desde la raíz del proyecto:

npm install

# Ejecutar la aplicación

desde la terminal ejecuta:

npm run dev


# Ejecutar tests

- Ejecutar todos los tests:

npm test

- Ejecutar tests con coverage:

npm run test-c
